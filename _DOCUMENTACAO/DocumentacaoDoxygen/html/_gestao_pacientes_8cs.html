Detailed Description

Classe de serviços para a gestão dos pacientes no sistema.

Este ficheiro contém a implementação da classe GestaoPacientes, que é responsável pela gestão dos pacientes no sistema, incluindo operações como adicionar, eliminar, verificar e obter pacientes. A classe interage com o repositório de pacientes para realizar estas operações.

Exceções lançadas:

  • ArgumentNullException quando algum dos parâmetros passados para os métodos é nulo.
  • MedicoNaoAutorizadoException se o médico não tiver autorização para realizar a operação.
  • EntidadeJaExisteException se o paciente a adicionar já existir.
  • KeyNotFoundException se o paciente a obter ou a eliminar não for encontrado.

Detailed Description

Classe para representar um médico, herdando de Pessoa.

A classe Medico representa um médico no sistema e herda as propriedades básicas da classe Pessoa. Inclui atributos específicos, como especialidade e número de ordem, além de métodos para comparar instâncias e gerar códigos hash.

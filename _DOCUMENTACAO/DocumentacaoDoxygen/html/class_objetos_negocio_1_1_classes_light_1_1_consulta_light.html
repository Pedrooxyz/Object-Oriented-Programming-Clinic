Detailed Description

Classe Light para representação simplificada de uma consulta.

A classe ConsultaLight é usada para armazenar informações resumidas sobre uma consulta, como o ID da consulta, o ID do paciente, o ID do médico e a data da consulta. Ela é usada principalmente para operações em que não é necessário o uso completo dos dados da consulta.

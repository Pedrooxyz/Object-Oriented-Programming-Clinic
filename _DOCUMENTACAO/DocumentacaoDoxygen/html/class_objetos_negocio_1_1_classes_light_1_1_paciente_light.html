Detailed Description

Classe Light para representação simplificada de um paciente.

A classe PacienteLight herda da classe PessoaLight e adiciona informações adicionais relacionadas ao paciente, como o número de utente. A classe é utilizada para operações rápidas, contendo apenas os dados essenciais do paciente.

Detailed Description

Classe Light que representa informações básicas sobre uma pessoa.

A classe PessoaLight contém apenas os dados essenciais sobre uma pessoa, como o ID e o nome. Esta classe é utilizada para operações simplificadas onde apenas informações básicas são necessárias.

bool RegrasNegocio.Interfaces.IGestaoConsultas.AdicionarConsulta ( ConsultaLight consultaLight,
MedicoLight medicoLight )

Adiciona uma nova consulta ao sistema.

Este método cria uma nova consulta com base nas informações fornecidas, e a adiciona ao repositório de consultas se o médico estiver autorizado. Caso a consulta já exista ou o médico não tenha permissão, o método retorna false.

Parameters
consultaLightObjeto que contém as informações mínimas de uma consulta.
medicoLightObjeto que representa o médico que está a adicionar a consulta.
Returns
Retorna true se a consulta for adicionada com sucesso, false caso contrário.
Exceptions
ArgumentNullExceptionSe algum dos parâmetros for null.
MedicoNaoAutorizadoExceptionSe o médico não estiver autorizado a adicionar consultas.

Implemented in RegrasNegocio.Servicos.GestaoConsultas.

\doxysubsection{Detailed Description}
Classe de serviços para a gestão das consultas no sistema. 

Este ficheiro contém a implementação da classe {\ttfamily Gestao\+Consultas}, que é responsável pela gestão das consultas no sistema, incluindo operações como adicionar, eliminar, verificar e obter consultas. A classe também inclui verificações de autorização para médicos e pacientes, lançando exceções apropriadas quando necessário. A classe interage com o repositório de consultas para realizar estas operações.

Exceções lançadas\+:
\begin{DoxyItemize}
\item {\ttfamily Medico\+Nao\+Autorizado\+Exception} quando o médico não está autorizado a realizar certas operações.
\item {\ttfamily Argument\+Null\+Exception} quando algum dos parâmetros passados para os métodos é nulo. 
\end{DoxyItemize}

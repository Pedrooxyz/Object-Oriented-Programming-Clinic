\doxysubsection{Detailed Description}
Classe de serviços para a gestão dos diagnósticos no sistema. 

Este ficheiro contém a implementação da classe {\ttfamily Gestao\+Diagnosticos}, que é responsável pela gestão dos diagnósticos no sistema, incluindo operações como adicionar, eliminar, verificar e obter diagnósticos. A classe interage com o repositório de diagnósticos para realizar estas operações.

Exceções lançadas\+:
\begin{DoxyItemize}
\item {\ttfamily Argument\+Null\+Exception} quando algum dos parâmetros passados para os métodos é nulo.
\item {\ttfamily Medico\+Nao\+Autorizado\+Exception} quando o médico não tem permissão para realizar a ação.
\item {\ttfamily Entidade\+Ja\+Existe\+Exception} quando a entidade a ser adicionada já existe.
\item {\ttfamily Colecao\+Nao\+Inicializada\+Exception} quando a coleção de diagnósticos não foi inicializada.
\item {\ttfamily Key\+Not\+Found\+Exception} quando um diagnóstico não é encontrado. 
\end{DoxyItemize}

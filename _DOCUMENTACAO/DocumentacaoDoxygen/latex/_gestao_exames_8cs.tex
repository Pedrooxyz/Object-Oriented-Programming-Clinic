Classe de serviços para a gestão dos exames no sistema. 

Este ficheiro contém a implementação da classe {\ttfamily Gestao\+Exames}, que é responsável pela gestão dos exames no sistema, incluindo operações como adicionar, eliminar, verificar e obter exames. A classe interage com o repositório de exames para realizar estas operações.

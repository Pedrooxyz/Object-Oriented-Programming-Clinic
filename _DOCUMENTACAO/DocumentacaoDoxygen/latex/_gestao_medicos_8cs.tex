Classe de serviços para a gestão dos pacientes no sistema. 

Este ficheiro contém a implementação da classe {\ttfamily Gestao\+Pacientes}, que é responsável pela gestão dos pacientes no sistema, incluindo operações como adicionar, eliminar, verificar e obter pacientes. A classe interage com o repositório de pacientes para realizar estas operações.

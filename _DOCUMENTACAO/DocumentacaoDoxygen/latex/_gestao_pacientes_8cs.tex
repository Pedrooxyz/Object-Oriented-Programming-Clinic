\doxysubsection{Detailed Description}
Classe de serviços para a gestão dos pacientes no sistema. 

Este ficheiro contém a implementação da classe {\ttfamily Gestao\+Pacientes}, que é responsável pela gestão dos pacientes no sistema, incluindo operações como adicionar, eliminar, verificar e obter pacientes. A classe interage com o repositório de pacientes para realizar estas operações.

Exceções lançadas\+:
\begin{DoxyItemize}
\item {\ttfamily Argument\+Null\+Exception} quando algum dos parâmetros passados para os métodos é nulo.
\item {\ttfamily Medico\+Nao\+Autorizado\+Exception} se o médico não tiver autorização para realizar a operação.
\item {\ttfamily Entidade\+Ja\+Existe\+Exception} se o paciente a adicionar já existir.
\item {\ttfamily Key\+Not\+Found\+Exception} se o paciente a obter ou a eliminar não for encontrado. 
\end{DoxyItemize}

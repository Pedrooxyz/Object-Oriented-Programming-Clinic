\doxysubsection{Detailed Description}
Define a interface para a gestão de consultas no sistema. 

Este ficheiro contém a interface {\ttfamily IGestao\+Consultas}, que especifica os métodos necessários para a adição, eliminação, verificação e obtenção de consultas, bem como a exportação de dados para ficheiros binários.

A interface foi projetada para garantir a separação de preocupações e permitir implementações consistentes, promovendo boas práticas de programação.

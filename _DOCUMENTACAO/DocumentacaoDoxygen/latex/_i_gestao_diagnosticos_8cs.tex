\doxysubsection{Detailed Description}
Define a interface para a gestão de diagnósticos no sistema. 

Este ficheiro contém a interface {\ttfamily IGestao\+Diagnosticos}, que especifica os métodos necessários para a adição, eliminação, verificação, obtenção e associação de diagnósticos, bem como a exportação de dados para ficheiros binários.

A interface foi projetada para garantir uma separação clara de preocupações e permitir implementações consistentes e robustas, promovendo boas práticas de programação.

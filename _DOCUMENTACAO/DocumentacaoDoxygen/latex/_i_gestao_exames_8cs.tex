Define a interface para a gestão de exames no sistema. 

Este ficheiro contém a interface {\ttfamily IGestao\+Exames}, que especifica os métodos necessários para realizar operações de gestão de exames, como adicionar, eliminar, atualizar e obter exames, bem como exportar dados para ficheiros binários.

A interface foi projetada para garantir boas práticas de programação, promovendo a separação de preocupações e a consistência entre diferentes implementações.

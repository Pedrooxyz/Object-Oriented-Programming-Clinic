Define a interface para a gestão de pacientes no sistema. 

Este ficheiro contém a interface {\ttfamily IGestao\+Pacientes}, que especifica os métodos essenciais para realizar operações de gestão de pacientes, como adicionar, eliminar, verificar, obter e associar dados relacionados a pacientes.

A interface promove a separação de responsabilidades e permite a implementação de diferentes estratégias de gestão de pacientes.

\doxysubsection{Detailed Description}
Classe para representar um exame realizado por um paciente. 

A classe \doxylink{class_dados_1_1_entidades_1_1_exame}{Exame} é usada para armazenar informações sobre exames realizados por pacientes, incluindo dados como tipo de exame, resultado, custo, e a data em que o exame foi realizado.

\doxysubsection{Detailed Description}
Classe para representar um médico, herdando de \doxylink{class_dados_1_1_entidades_1_1_pessoa}{Pessoa}. 

A classe \doxylink{class_dados_1_1_entidades_1_1_medico}{Medico} representa um médico no sistema e herda as propriedades básicas da classe \doxylink{class_dados_1_1_entidades_1_1_pessoa}{Pessoa}. Inclui atributos específicos, como especialidade e número de ordem, além de métodos para comparar instâncias e gerar códigos hash.

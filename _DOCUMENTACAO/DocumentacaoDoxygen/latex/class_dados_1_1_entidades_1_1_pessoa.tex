Classe base para representar uma pessoa. 

A classe \doxylink{class_dados_1_1_entidades_1_1_pessoa}{Pessoa} é utilizada para representar uma entidade básica com informações como ID, nome, género e data de nascimento. Inclui métodos para manipular e comparar objetos deste tipo.

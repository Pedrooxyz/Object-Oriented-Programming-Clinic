\doxysubsection{Detailed Description}
Classe responsável pela gestão de consultas num repositório. 

A classe {\ttfamily \doxylink{class_dados_1_1_repositorios_1_1_consultas_repositorio}{Consultas\+Repositorio}} fornece funcionalidades para manipular consultas associadas a pacientes. As operações incluem adicionar, remover, verificar a existência e obter consultas específicas ou todas as consultas associadas a pacientes.

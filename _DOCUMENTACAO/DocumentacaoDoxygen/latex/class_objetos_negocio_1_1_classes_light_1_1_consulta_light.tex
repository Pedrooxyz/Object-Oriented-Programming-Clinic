\doxysubsection{Detailed Description}
Classe Light para representação simplificada de uma consulta. 

A classe {\ttfamily \doxylink{class_objetos_negocio_1_1_classes_light_1_1_consulta_light}{Consulta\+Light}} é usada para armazenar informações resumidas sobre uma consulta, como o ID da consulta, o ID do paciente, o ID do médico e a data da consulta. Ela é usada principalmente para operações em que não é necessário o uso completo dos dados da consulta.

\doxysubsection{Detailed Description}
Classe Light para representação simplificada de um exame. 

A classe {\ttfamily \doxylink{class_objetos_negocio_1_1_classes_light_1_1_exame_light}{Exame\+Light}} é usada para armazenar informações resumidas sobre um exame, como o ID do exame, o tipo de exame, a data em que o exame foi realizado e o ID do paciente associado. Ela é usada principalmente para operações em que não é necessário o uso completo dos dados do exame.

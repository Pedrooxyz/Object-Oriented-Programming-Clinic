\doxysubsection{Detailed Description}
Classe Light para representação simplificada de um paciente. 

A classe {\ttfamily \doxylink{class_objetos_negocio_1_1_classes_light_1_1_paciente_light}{Paciente\+Light}} herda da classe {\ttfamily \doxylink{class_objetos_negocio_1_1_classes_light_1_1_pessoa_light}{Pessoa\+Light}} e adiciona informações adicionais relacionadas ao paciente, como o número de utente. A classe é utilizada para operações rápidas, contendo apenas os dados essenciais do paciente.
